<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
* Python

### Installation

1. Clone the Front-end repo
   ```sh
   git clone https://github.com/alexlowe7/React-Translator.git
   ```
2. In the project directory run:
   ```sh
   npm start
   ```
3. Clone the Back-end repo in a new terminal tab
   ```sh
   git clone https://github.com/alexlowe7/React-Translator-Backend.git
   ```
4. Install dependencies:
   ```sh
   pipenv install
   ```
5. Activate virtual environment:
   ```sh
   pipenv shell
   ```
6. Move into the 'backend' directory
   ```sh
   cd backend
   ```
7. Run server
   ```sh
   python manage.py runserver
   ```
8. Start translating! http://localhost:3000/
